# Processing Art

[Processing](https://www.processing.org/download/) sketches I made during a Creative Code course

## Getting Started

### Prerequisities

You only need:
```
* Java 1.8
* Processing Software
```

### Running it!

* Open the .pde file with Processing and press the 'Play' button on the top left to compile and run the selected sketch
